# Main Space Archive

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何单个域名或具体网站，主要目的是将不同来源、不同用途的 HTML 页面集中存放，便于管理和访问。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放所有归档 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本说明文件
```

## 页面归档说明

- 所有归档页面统一放置在 `pages/` 目录下，文件名按顺序或主题命名。
- 每个页面均为独立完整的 HTML 文件，可单独在浏览器中打开。
- 若页面依赖公共资源（如样式、脚本），请确保资源路径正确引用 `assets/` 目录中的文件。
- 归档页面不包含任何隐私数据、敏感信息或追踪代码。

## 维护说明

- 新增页面时，请将 HTML 文件放入 `pages/` 目录，并更新 `index.html` 中的导航链接。
- 修改现有页面时，请保持文件结构稳定，避免破坏已有链接。
- 删除页面时，请同步移除 `index.html` 中的对应入口。
- 所有提交应保持页面内容合规、无恶意代码。

## 许可

本项目采用通用许可，不对归档内容作额外限制。请遵守相关法律法规。
